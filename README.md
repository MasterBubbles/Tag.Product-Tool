# Tag.Product Tool

## Usage

### GUI Version
Download the latest relase.
Either run TagProductTool.exe or drag a byml or json on it.
Select the output folder.

### CMD Version
`TagProductTool.exe INPUT_FILE OUTPUT_PATH`
